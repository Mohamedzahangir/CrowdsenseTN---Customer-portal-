# 🚍 CrowdSense TN – Customer Portal

CrowdSense TN Customer Portal is a smart public transportation application that enables passengers to make informed travel decisions through real-time transit insights, crowd awareness, ETA visibility, and intelligent route discovery.

## 🌐 Live Demo

https://crowdsense-tn-customer-portal.vercel.app/#/home

---

# Overview

The Customer Portal serves as the passenger-facing interface of the CrowdSense TN ecosystem.

It provides a modern digital transit experience by helping commuters:

* Discover bus routes
* Track buses in real time
* Monitor crowd levels before boarding
* View estimated arrival times (ETA)
* Receive travel notifications
* Save frequently used routes
* Analyze personal travel statistics

The application is designed to improve public transportation accessibility, convenience, and passenger confidence.
